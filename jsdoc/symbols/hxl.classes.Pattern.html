hxl.classes.Pattern(tag, include_attributes, exclude_attributes)
A compiled pattern for matching HXL columns.

You should not normally call this constructor directly. All functions that use a pattern compile it as needed, so you can stick with strings. If you want to precompile a pattern, use the static #parse method.

The other useful method in this class is #match, with tests the pattern against a hxl.classes.Column.

A pattern specification looks like a normal HXL hashtag and attributes, except that it can contain "-" attributes as well as "+" attributes. For example, the following pattern matches any column with the tag "#org", the attribute "+funder", and *not* the attribute "+impl" (any other attributes are ignored):

#org+funder-impl

More examples:

#adm1 will match the following: #adm1, #adm1+code, #adm1+fr, #adm1+name, etc.

#adm1-code will match #adm1, #adm1+fr, and #adm1+name, but *not* #adm1+code (because the +code attribute is explicitly forbidden).

#adm1+name-fr will match #adm1+name or #adm1+name+en, but not #adm1+name+fr

Parameters:
{string} tag
The basic tag to use in the pattern, with the leading '#'
{array} include_attributes
A (possibly-empty) list of attribute names that must be present, without the leading '+'
{array} exclude_attributes
A (possibly-empty) list of attribute names that must *not* be present, without the leading '+'

{boolean} match(column)
Test if a column matches this pattern. 
data.columns.forEach(function(col) {
  if (pattern.match(col)) {
    console.log("Found a match: " + col.displayTag);
  }
});
This method is used heavily in the filter classes.
Parameters:
{hxl.classes.Column} column
The column to test.
Returns:
{boolean} true on match, false otherwise.
<static> hxl.classes.Pattern.parse(pattern, useException)
Parse a string into a tag pattern. 
var pattern = hxl.classes.Pattern.parse('#org+funder-impl');
It is safe to pass an already-compiled hxl.classes.Pattern to this method; it will simple be returned as-is.
Parameters:
{string} pattern
a tag-pattern string, like "#org+funder-impl"
useException
(optional) throw an exception on failure.
Returns:
a hxl.classes.Pattern, or null if parsing fails (and useException is false).
